Command Palette

The ⌘K skill launcher — OneBrain runs on /skill slash-commands, so this is the operator-console heart. One reusable component (.command-palette / .cmdk-* in components.css) driven by command-palette.js from a declarative config. Press ⌘K / Ctrl K, type to filter, ↑↓ to navigate, ↵ to run.

// Source basis — real onebrain-ai/website modules behind the console chrome

Every preview in this package is extracted from real source, not invented. The operator console's status chrome is fed by five modules in onebrain-ai/website · src/ (preserved in this package under source_examples/) — modeled below as live design-system cards, each rendering the exact chip, badge, gate, or header set it produces in production.

lib · build

version

Build-time fetch of the latest CLI release tag from the GitHub Releases API, memoized at module scope and sanitized to a semver-safe charset. Falls back to a known-good string when the API is unreachable, so the version chip never renders empty.

lib · build

gh-stars

Build-time star count for onebrain-ai/onebrain, baked into static HTML. formatStars() compacts the number; a 0 result hides the badge entirely — "0 stars" is worse social proof than none.

lib · waitlist

disposable-domains

Frozen Set blocklist (~150 throwaway domains) gating the waitlist form. On a match the server returns 200 OK with no write — the bot can't tell "blocked" from "accepted", so it stops retrying and sender reputation stays clean.

middleware

Astro edge middleware that attaches security headers to dynamic API responses, and catches downstream throws so the headers still ship on a 500 instead of leaking a bare error.

lib · ssr

brain-id

Per-SSR-pass counter so each <Brain /> render gets distinct gradient/filter ids without Math.random — keeping the HTML byte-stable so edge caching isn't broken on every request.
