// Source basis — icons fronting the website's real modules (onebrain-ai/website)
  • version.ts — ob-clock fronts the build-time CLI version fetched from the GitHub Releases API; falls back to 3.0.0 offline, memoized so Hero + Footer share one fetch.
  • gh-stars.ts — ob-skill marks the star badge baked in at build; returns 0 on any failure and the Cloud badge stays hidden on 0 (no "0 stars").
  • disposable-domains.ts — ob-x is the block mark for the ~150-domain waitlist blocklist; a match returns 200 OK with no write, so the bot can't tell blocked from accepted.
  • brain-id.ts — ob-brain is the wordmark; the module hands each <Brain/> a distinct SSR gradient id from a counter (no Math.random, so HTML stays edge-cacheable).
  • middleware.ts — ob-harness for the system layer that adds API security headers (HSTS, nosniff, X-Frame-Options: DENY, strict-origin, COOP: same-origin) — shipped even on a 500.
